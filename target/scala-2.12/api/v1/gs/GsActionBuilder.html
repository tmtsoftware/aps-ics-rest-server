GsActionBuilder 

class GsActionBuilder extends ActionBuilder[GsRequest, AnyContent] with RequestMarkerContext with HttpVerbs

The action builder for the Galil resource.

This is the place to put logging, metrics, to augment the request with contextual data, and manipulate the result.
